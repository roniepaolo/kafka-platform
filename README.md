
# Kafka Cluster 3.3.x in KRaft Mode

This project is a deployment of a Kafka Cluster, Kafka Connect and Schema Registry using Terraform, Ansible and Docker.


## Features

- 3 Brokers
- 1 Producer
- 1 Consumer
- 1 Schema Registry
